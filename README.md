# Ms Team Website

Website resmi untuk Ms Team, tim developer dan cyber security yang fokus pada secure engineering, offensive testing, dan automation.

## Deskripsi

Ms Team adalah unit khusus yang membangun sistem digital yang cepat, aman, dan siap bertahan dari serangan. Kami mengkhususkan diri dalam pengembangan perangkat lunak yang aman, pengujian keamanan ofensif, dan otomasi proses.

## Fitur

- **Secure Development**: Pengembangan aplikasi dengan pendekatan keamanan pertama.
- **Offensive Testing**: Pengujian keamanan sistem dan aplikasi.
- **Automation**: Otomasi proses untuk efisiensi dan keamanan.
- **Team Showcase**: Profil anggota tim dengan spesialisasi masing-masing.

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Deployment**: Vercel
- **Server**: Node.js (untuk development lokal)

## Instalasi dan Menjalankan Lokal

1. Clone repository ini:
   ```bash
   git clone https://github.com/ms-onion/web.git
   cd web
   ```

2. Install dependencies (jika ada):
   ```bash
   npm install
   ```

3. Jalankan server lokal:
   ```bash
   node server.js
   ```

4. Buka browser dan akses `http://localhost:7634`

## Deployment

Proyek ini sudah dikonfigurasi untuk deployment otomatis ke Vercel. Setiap push ke branch main akan memicu deployment baru.
